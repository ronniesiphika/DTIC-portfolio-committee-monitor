# Portfolio Committee Monitor

This project monitors a source link for portfolio committee content, stores snapshots, detects changes, analyzes newly communicated information, and produces daily reports.

## Features
- Fetch source content
- Save raw snapshots
- Parse and clean content
- Compare with prior version
- Generate Markdown and JSON reports
- Run daily using GitHub Actions
